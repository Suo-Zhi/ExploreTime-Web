# Explore Time

**简介：**

一个基于知识模块化的输出式学习系统（前端部分）。

**启动项目：**

```
npm i
```

```
npm run dev
```

**技术栈：**

-   前端：Vue3 + TypeScript + Vite + Pinia + ArcoVue
-   后端：NestJS + Prisma + PostgreSQL
